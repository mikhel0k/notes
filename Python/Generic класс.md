Чтобы работать с Generic классами необходимо создавать чекер типов TypeVar [[TypeVar]]
```
from typing import TypeVar, Generic

ModelType = TypeVar("ModelType", bound=BaseModel)
```

Потом этот TypeVar нужно передать в класс Generic от которого от наследоваться уже в создаваемом классе
```
class Class(Generic[ModelType])
```

Также чтобы все полноценно работало в инициализаторе надо принять класс с которым будем работать
```
def __init__(self, model: Type[ModelType]):
        self.model = model
```

Generiv класс это такой класс который унифицировано работает с разными типами, а с каким конкретно типом он будет работать мы указываем позже в процессе создания объектов этого класса.
