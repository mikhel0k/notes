#полиморфизм #Python #типы_данных #декларация_типов
TypeVar это декларация типов которая делает проверку тот ли это класс который мы ждем. Вся прелесть этой переменной что благодаря ней соблюдается полиморфизм. То-есть я могу указать родительский класс в качестве ожидаемого и любой дочерний класс пройдет проверку.

```
from typing import TypeVar

ModelType = TypeVar("ModelType", bound=BaseModel)
```

Вот как раз в bound мы передаем родительский класс чтобы потом в эту переменную можно было записывать дочерние классы. А "ModelType" это строчка для отладочной информации.
