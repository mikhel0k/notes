Запрос на создание записи в базе данных делается следующим образом:
Объявляется асинхронная функция в которую передаются аргументы: pydantic схема для создания записи в базе данных, в ней не должен упоминаться id и будет передаваться информация о содержимом записи, а также объект AsyncSession который будет являться сессией в базе данных.
```
async def book_create(  
        book: BookCreate,  
        session: AsyncSession,  
):
```

Потом распаковываем информацию из схемы, создаем объект записи в базе данных и добавляем эту запись в сессию
```
db_book = Book(**book.model_dump())  
session.add(db_book)
```

После чего асинхронно работаем с базой данных
```
await session.commit()  
await session.refresh(db_book)  
return db_book
```
Через commit применяем все сделанные  внутри сессии изменения в базу данных. Потом синхронизируем запись в бд с переменной в программе и возвращаем эту измененную переменную из функции.
