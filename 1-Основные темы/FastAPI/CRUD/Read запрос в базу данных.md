Запрос на получение записи из базы данных делается следующим образом
Объявляется асинхронная функция в которую передается какой-то параметр по которому будет вестись поиск элементов, зачастую это либо id записи в базе данных, либо какое-то другое поле, а также вторым аргументов получается асинхронная сессия из базы данных.
```
async def read_book(
		book_id: int,
		session: AsyncSession
):
```

Если делается запрос на поиск только по первичному ключу, то можно использовать следующую конструкцию:
```
author = await session.get(Author, author_id)
return author
```

Но в случае, если мы хотим сделать поиск по полю не являющемуся первичным ключом, то используется конструкция похожая на SQL запрос
```
stmt = select(Author).where(Author.first_name == author_name)  
answ = await session.execute(stmt)
```
В переменную stmt записывается запрос на поиск и через session.execute выполняется в базе данных.

Если планируется что будет найдено несколько записей, то используется конструкция 
```
authors = answ.scalars().all()
```
Если планируется получить первую запись
```
authors = answ.scalars().first()
```

Если планируется только одна запись
```
author = answ.scalar_one_or_none()
```
Также в случае, если нужно обязательно получить одну запись
```
author = answ.scalar_one()
```
