Запрос на удаление записи в базе данных делается следующим образом:
Объявляется асинхронная функция в которую передаются аргументы: id объекта который надо будет удалить, а также объект AsyncSession который будет являться сессией в базе данных.
```
async def delete_author(  
        author_id: int,  
        session: AsyncSession  
):
```

Для того чтобы удалить запись из базы данных, необходимо сначала ее найти.
```
author = await session.get(Author, author_id)
```

После чего удалить и сохранить изменения в базе данных.
```
await session.delete(author)  
await session.commit()
```
