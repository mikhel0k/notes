
Не рекомендуется использовать онлайн генераторы так как допускается возможность утечки информации. Чтобы ее избежать надо использовать локальные генераторы. Рассмотрю генерацию на примере openssl.

Чтобы сгенерировать приватный ключ необходимо выполнить команду 
```
openssl genrsa -out jwt-private.pem 2048
```

Чтобы сгенерировать публичный ключ на основании приватного надо выполнить команду
```
openssl rsa -in jwt-private.pem -outform PEM -pubout -out jwt-public.pem
```
