```
from passlib.context import CryptContext  
  
  
pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")  
  
  
def get_password_hash(password):  
    return pwd_context.hash(password)  
  
  
def verify_password(plain_password, hashed_password):  
    return pwd_context.verify(plain_password, hashed_password)
```

При помощи этого кода можно хэшировать пароли, по синтаксису все просто, импортируем класс при помощи которого задаем контекст хэширования, в schemes передаем каким алгоритмом будем пользоваться, параметр deprecated ставится auto чтобы автоматически отсеивать устаревшие версии алгоритмов. Ну и следующие две функции это просто оболочки для функций этого класса, чтобы более удобно их использовать из других мест в коде. get_password_hash получает хэш строки передаваемой в нее, а verify password сравнивает строку которую получается и хэш уже имеющийся.
