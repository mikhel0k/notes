#библиотеки #FastAPI #postgresql 

```
pip install "fastapi[uvicorn]"
```
Устанавливаем библиотеку fastapi и движок uvicorn, который нужен будет чтобы запускать локальный сервер.

```
pip install python-multipart
```
Необходима для обработки форм, а также упрощает загрузку файлов от пользователей.

```
pip install sqlalchemy  
```
Данная библиотека нужна для работы с базами данных, основной функционал это переделывать Python классы в таблицы в бд.

```
pip install "databases[postgresql]" 
```
Нужна для асинхронной работы с postgresql можно в квадратных кавычках указать любую другую бд.

```
pip install "python-jose[cryptography]"
```
Эта необходима чтобы шифровать и дешифровывать JWT токены.

```
pip install "passlib[bcrypt]"
```
Нужна для хэширования пароля.

```
pip install pydantic
```
Данная библиотека нужна чтобы прописывать схемы входящих и исходящих в API запросов, а также для того чтобы валидировать все входящие и исходящие запросы.

```
pip install email-validator
```
Нужна для валидации указанных полей в pydantic схемах как email.

```
pip install alembic
```
Нужна для создания миграций баз данных при помощи которых можно изменять состояние таблиц, создавать новые таблицы и связи и откатывать все изменения.

```
pip install python-dotenv
```
Данная библиотека помогает читать файл конфигурации .env.

```
pip install pydantic-settings 
```
А эта создает класс для более удобного взаимодействия с настройками
