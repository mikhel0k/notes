Фикстуры это заранее написанные именованные участки кода, по сути просто самые обычные функции либо данные, только подготовленные под специфическую работу с тестами.
Существует несколько видов фикстур:
- фикстуры данных
  ```
  @pytest.fixture
def sample_user():
    return {"name": "John", "age": 30}

	def test_user_age(sample_user):
	    assert sample_user["age"] == 30
  ```

- фикстуры ресурсов(например базы данных)
	```
	@pytest.fixture
	def db_connection():
	    conn = connect_to_db()
	    yield conn
	    conn.close()  # Важно освобождать ресурсы!
	```

- фикстуры моки [[Mok-заглушка]]
  ```
	@pytest.fixture
	def mock_external_api():
	    with patch('requests.get') as mock:
	        mock.return_value.json.return_value = {"status": "ok"}
	        yield mock
	```
